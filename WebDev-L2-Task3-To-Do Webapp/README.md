# To-Do Web App

## About
I developed this To-Do Web App using HTML, CSS and JavaScript. It helps users manage their daily tasks by adding, editing, completing and deleting tasks. The data is stored using Local Storage.

## Features
- Add Task
- Edit Task
- Delete Task
- Mark Task as Complete
- Pending and Completed Tasks
- Task Counter
- Date and Time
- Local Storage Support
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage
