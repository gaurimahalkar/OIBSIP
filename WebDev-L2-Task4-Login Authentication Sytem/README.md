# Login Authentication System

## About
I created this Login Authentication System using HTML, CSS and JavaScript. Users can register, log in and access the dashboard after successful login. Passwords are secured using SHA-256 hashing and user data is stored in Local Storage.

## Features
- User Registration
- User Login
- Email Validation
- Password Validation
- Duplicate Email Check
- SHA-256 Password Hashing
- Local Storage
- Dashboard Page
- Logout

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage
- SHA-256
