# Calculator

## About
I created this calculator using HTML, CSS and JavaScript. It can perform basic mathematical operations with a simple and easy-to-use interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Decimal Support
- Clear Button
- Delete Button
- Error Message for Invalid Input
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript
